# NocLauncher

Кастомный лаунчер Minecraft с упором на практичность: **Java + Bedrock**, оффлайн-first, диагностика и удобные настройки производительности.

## Что умеет

- Java-режим: vanilla / Forge / OptiFine
- Bedrock-режим
- Оффлайн-запуск (онлайн-авторизация временно отключена)
- Автосохранение ника
- Оффлайн-скины (файл / ник / URL / авто)
- Автодобавление сервера (в профиль лаунчера и `%APPDATA%/.minecraft`)
- Диагностика запуска и логи
- FPS boost preset'ы + JVM tuning
- Режим экономии ресурсов UI (low-power)
- При запуске игры лаунчер сворачивается/уходит в фон и возвращается после выхода

## Скачать (Release Assets)

Релиз: `v1.0.0`

- Windows x64: `NocLauncher-1.0.0-windows-x64.exe`
- Windows x86: `NocLauncher-1.0.0-windows-x86.exe`
- Linux x64: `NocLauncher-1.0.0-linux-x64.AppImage`
- macOS: в процессе выката (см. Actions)

## Быстрый старт

1. Скачайте установщик/сборку под вашу платформу из GitHub Releases.
2. Установите/запустите NocLauncher.
3. Выберите режим (**JAVA** или **BEDROCK**).
4. Для Java: выберите версию и тип загрузчика.
5. Введите ник и нажмите **Играть**.
