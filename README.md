# NocLauncher

Кастомный лаунчер Minecraft с упором на практичность: **Java + Bedrock**, быстрый запуск, оффлайн-режим, диагностика и удобные настройки.

## Что внутри

- Java-режим: vanilla / Forge / OptiFine
- Bedrock-режим
- Оффлайн-запуск и сохранение ника
- Диагностика запуска и логов
- Автодобавление сервера
- Тонкие настройки загрузки и JVM
- Режим экономии ресурсов лаунчера
- Космический UI/фон (можно отключить)

## Скачать

- Готовый установщик: `release/NocLauncher Setup 1.0.0.exe`

## Быстрый старт

1. Скачайте `.exe` из папки `release/` или из релизов.
2. Установите NocLauncher.
3. Откройте лаунчер, выберите версию и режим загрузчика.
4. Введите ник и нажмите **Играть**.

## Настройки, которые чаще всего нужны

- **Отключить анимированный фон** — снижает нагрузку на GPU.
- **Закрывать лаунчер после запуска игры** — чтобы не висел в фоне.
- **JVM preset / RAM** — для стабильности на слабых/сильных ПК.

## Сборка из исходников

```bash
npm install
npm run build:win
```

Результат сборки:

`dist/NocLauncher Setup 1.0.0.exe`
