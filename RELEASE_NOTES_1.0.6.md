# NocLauncher v1.0.6

## Что нового
- Обновлён UI и визуальная часть лаунчера (иконки, карточки, оформление боковой панели).
- Доработаны кнопки выбора режимов Java / Bedrock.
- Улучшен Bedrock launch-flow:
  - если Minecraft Bedrock не установлен — лаунчер сразу открывает Microsoft Store;
  - лишние repair/replacement операции не запускаются, когда Bedrock отсутствует.
- Улучшена логика фоновых операций и уменьшено количество лишних окон при запуске.
- Обновлён набор анти-тамперов/проверок и восстановлений для Bedrock.
- Повышена стабильность предпросборочного и релизного поведения.

## Важно (Windows Defender / SmartScreen)
При первом запуске Windows может показывать предупреждение SmartScreen/Defender.
Это возможно из-за отсутствия коммерческой цифровой подписи у сборки, а не обязательно из-за вредоносного кода.

Рекомендуется:
1. Скачивать сборки только из официального GitHub Releases.
2. Проверять SHA256-хэш опубликованных файлов.
3. При необходимости дополнительно проверять файл в VirusTotal.
