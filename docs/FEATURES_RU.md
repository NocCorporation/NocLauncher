# Возможности NocLauncher

## Основные
- Java + Bedrock
- Vanilla / Forge / OptiFine
- Оффлайн-first режим (без MS auth на текущем этапе)
- Автосохранение ника

## Геймплей и контент
- Оффлайн-скины: файл / ник / URL / авто
- Автодобавление сервера в `servers.dat`
- Безопасный запуск (временно без модов)

## Стабильность
- Preflight-проверки перед запуском
- Диагностика запуска и логи
- Health timeline/телеметрия старта

## UX
- Компактный UI
- Скроллящиеся настройки
- Переключаемые визуальные эффекты
- Сценарий «запустил игру → лаунчер ушёл в фон → после выхода вернулся»

## Производительность
- FPS Boost mode
- Preset'ы оптимизации (`balanced` и др.)
- JVM tuning
- Low-power UI (отключение тяжёлого рендера)
