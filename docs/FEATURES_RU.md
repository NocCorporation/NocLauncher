# Возможности NocLauncher

## Основные
- Java + Bedrock
- Vanilla / Forge / OptiFine
- Оффлайн-режим запуска
- Сохранение ника

## Стабильность
- Проверки перед запуском
- Диагностика и логирование
- Режим безопасного запуска без модов
- Авто-синхронизация серверов

## UX
- Компактный UI
- Быстрый выбор версий
- Скроллящиеся настройки
- Переключаемый космический фон

## Оптимизация
- Режим экономии ресурсов (GPU saver)
- Управление RAM/JVM
- Настройки загрузки (скорость, параллельность, источник)
