# NocLauncher — полный гайд

## 1) Установка

1. Запустите `NocLauncher Setup 1.0.0.exe`.
2. Дождитесь окончания установки.
3. Откройте лаунчер.

## 2) Первый запуск

- Выберите режим: **JAVA** или **BEDROCK**.
- Для Java выберите версию и загрузчик (vanilla/forge/optifine).
- Введите ник.
- Нажмите **Играть**.

## 3) Рекомендуемые настройки

### Производительность
- Включите **Отключить анимированный фон (экономия ресурсов)**.
- Для слабых ПК выставьте JVM preset: **Слабый ПК**.
- Если есть лаги — уменьшите RAM до адекватного диапазона.

### Удобство
- Включите **Закрывать лаунчер после запуска игры**.
- Ник сохраняется автоматически между перезапусками.

## 4) Моды и ресурсы

- Раздел **Моды • Текстуры • Шейдеры** открывает библиотеку контента.
- Для проблемных сборок используйте **Безопасный запуск (временно без модов)**.

## 5) Логи и диагностика

- Кнопка **Логи** показывает диагностическую информацию.
- При ошибках запуска отправляйте лог целиком в Issue.
