# NocLauncher — полный гайд

## 1) Установка

### Windows
1. Скачайте `.exe` из Release Assets.
2. Запустите установщик.
3. Откройте NocLauncher.

### Linux
1. Скачайте `.AppImage`.
2. Выдайте право на запуск:
   ```bash
   chmod +x NocLauncher-1.0.0-linux-x64.AppImage
   ./NocLauncher-1.0.0-linux-x64.AppImage
   ```

### macOS (beta)
- Используйте сборку под свою архитектуру (x64/arm64).
- Если Gatekeeper блокирует, откройте через ПКМ → Open.

## 2) Первый запуск

- Выберите режим: **JAVA** или **BEDROCK**.
- Для JAVA выберите версию и тип загрузчика.
- Введите ник.
- Нажмите **Играть**.

## 3) Рекомендуемые настройки (по умолчанию уже оптимизированы)

- `FPS Boost Mode`: ВКЛ
- `FPS preset`: `balanced`
- `Low-power UI`: ВКЛ
- Поведение лаунчера при старте игры: уход в фон и возврат после выхода

## 4) JAVA: Vanilla / Forge / OptiFine

- Начинайте с Vanilla для проверки базовой стабильности.
- Затем включайте Forge/OptiFine под нужную версию.
- При конфликте модов используйте безопасный запуск (временно без модов).

## 5) Оффлайн-режим и ник

- Онлайн auth сейчас отключён intentionally.
- Ник сохраняется автоматически между сессиями.

## 6) Скины

Оффлайн-скины поддерживают:
- файл,
- ник,
- URL,
- авто-режим.

## 7) Серверы

Лаунчер автоматически добавляет сервер в `servers.dat` (локальный профиль + `%APPDATA%/.minecraft`).

## 8) Диагностика

- Используйте раздел **Логи**.
- Для баг-репорта отправляйте лог + сценарий + версию.
