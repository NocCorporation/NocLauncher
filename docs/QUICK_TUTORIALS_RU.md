# NocLauncher — мини-гайды и туториалы (база)

Ниже только самое нужное: как быстро запустить и где крутить ключевые настройки.

## 1) Первый запуск за 2 минуты

1. Открой NocLauncher.
2. Выбери режим:
   - **JAVA** — классический Minecraft Java Edition
   - **BEDROCK** — Bedrock-режим
3. Для JAVA выбери версию игры.
4. Введи ник.
5. Нажми **Играть**.

## 2) Как выбрать тип сборки (Vanilla / Forge / OptiFine)

- **Vanilla** — чистая игра, минимум проблем.
- **Forge** — если нужны моды.
- **OptiFine** — если нужен прирост FPS и графические настройки.

Совет: при проблемах сначала проверяй запуск на **Vanilla**, затем возвращай Forge/OptiFine.

## 3) Как включить максимум стабильности

Открой настройки и проверь:

- **FPS Boost Mode**: ВКЛ
- **FPS preset**: `balanced` (по умолчанию рекомендован)
- **Low-power UI / отключение анимаций**: ВКЛ
- **Безопасный запуск (без модов)**: включать только для диагностики

## 4) Как работает поведение лаунчера при старте игры

При запуске игры лаунчер **не завершается жёстко**:
- он уходит в фон/сворачивается,
- после выхода из игры возвращается.

Это сделано для стабильности и удобства.

## 5) Оффлайн-скины: быстрый сценарий

Доступные варианты:
- выбрать локальный файл скина,
- подтянуть скин по нику,
- подтянуть по URL,
- авто-режим.

Если скин не применился:
1. Проверь PNG формат,
2. Перезапусти игру,
3. Посмотри логи в лаунчере.

## 6) Автодобавление сервера

Если указан сервер, лаунчер добавляет его в:
- `<gameDir>/servers.dat`
- `%APPDATA%/.minecraft/servers.dat`

Если сервер не появился — проверь права на запись и запусти лаунчер от обычного пользователя (без конфликтных ограничений папки).

## 7) Где смотреть ошибки

- Кнопка/раздел **Логи** в лаунчере.
- Смотри последние строки перед падением.
- Для Issue отправляй:
  - шаги воспроизведения,
  - версию игры,
  - тип загрузчика (Vanilla/Forge/OptiFine),
  - лог.

## 8) Linux запуск (AppImage)

```bash
chmod +x NocLauncher-1.0.0-linux-x64.AppImage
./NocLauncher-1.0.0-linux-x64.AppImage
```

## 9) macOS запуск (beta без подписи)

Если macOS блокирует запуск:
- открой через **ПКМ → Open**,
- при необходимости сними quarantine:

```bash
xattr -dr com.apple.quarantine /Applications/NocLauncher.app
```

## 10) Что делать, если «просто не запускается»

1. Сначала попробуй Vanilla.
2. Отключи моды/шейдеры (безопасный запуск).
3. Проверь Java path (или очисти его для авто-выбора).
4. Отключи тяжёлые визуальные эффекты UI.
5. Сними и пришли лог.
