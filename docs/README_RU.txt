NocLauncher (Electron) — полностью рабочий лаунчер Minecraft

Функции:
- Запуск ванильных версий (релизы/снапшоты) через официальный Mojang manifest.
- Оффлайн-профиль по нику.
- Microsoft вход (Device Code flow) — удобно на ПК/телефоне.
- Настройки: папка игры, Java путь, RAM min/max.
- Логи и прогресс загрузки.

Требования:
- Windows 10/11 (или Linux/Mac).
- Node.js 18+ (лучше 20 LTS).
- Для Minecraft 1.18+ нужна Java 17+. (Для 1.20.5+ чаще всего Java 21.)

Запуск (разработка):
1) Открой папку проекта.
2) Установи зависимости:
   npm install
3) Запусти:
   npm start

Сборка под Windows (exe установщик):
1) npm install
2) npm run build:win
Готовое будет в папке dist/

Если Minecraft не запускается:
- Укажи путь к javaw.exe в Настройках (Temurin/Adoptium рекомендуются).
- Проверь интернет (первый запуск скачивает ассеты).
- Открой вкладку Логи и скинь текст — разберём.
