# Troubleshooting

## Лаунчер грузит GPU
- Включите: **Отключить анимированный фон (экономия ресурсов)**.
- Перезапустите лаунчер после смены настройки.

## Игра не стартует
- Проверьте Java Path (или очистите его, чтобы лаунчер выбрал сам).
- Попробуйте **Безопасный запуск (временно без модов)**.
- Проверьте логи через кнопку **Логи**.

## Вылеты на модах
- Убедитесь, что версия модов совпадает с версией Minecraft.
- Для OptiFine используйте стабильные сборки.

## Не сохраняется ник
- Ник сохраняется автоматически. Если не сохраняется — проверьте права на запись в папку лаунчера.
