# Troubleshooting (RU)

## 1) Лаунчер грузит GPU/CPU
- Включите **Low-power UI / отключение анимаций**.
- Перезапустите лаунчер.

## 2) Игра не стартует
1. Проверьте Java path (или очистите для авто-выбора).
2. Попробуйте Vanilla.
3. Включите безопасный запуск (без модов).
4. Посмотрите логи.

## 3) Вылеты на Forge/OptiFine
- Убедитесь, что версия модов совпадает с версией Minecraft.
- Проверьте совместимость конкретной сборки Forge/OptiFine.
- Для проверки запустите Vanilla.

## 4) Не применяется скин
- Проверьте формат PNG.
- Попробуйте другой источник (файл/ник/URL).
- Перезапустите игру.

## 5) Сервер не добавился
- Проверьте права записи в папки профиля.
- Убедитесь, что путь к gameDir корректный.
- Перезапустите лаунчер и повторите запуск.

## 6) Linux AppImage не запускается
```bash
chmod +x NocLauncher-1.0.0-linux-x64.AppImage
./NocLauncher-1.0.0-linux-x64.AppImage
```

## 7) macOS блокирует запуск
- Откройте через ПКМ → Open.
- Если нужно:
```bash
xattr -dr com.apple.quarantine /Applications/NocLauncher.app
```

## 8) Что прислать для диагностики
- Версия лаунчера
- ОС
- Версия Minecraft
- Тип загрузчика (Vanilla/Forge/OptiFine)
- Полный лог запуска
